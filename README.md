# 🚗 Cars API

API REST para gestionar carros (CRUD).

## ⚙️ Instalación

```bash
git clone https://github.com/SergioRamirez06/cars-api-nest
cd cars-api
npm install
```

## ▶️ Ejecutar

```bash
npm run start:dev
```

## 📡 Endpoints

* GET /cars
* GET /cars/:id
* POST /cars
* PATCH /cars/:id
* DELETE /cars/:id
